App for managing my librarly, segregating books, finding where they are exactly (shelf and rack)

Based on simple stack from (Roda-Sequel-Stack)[https://github.com/jeremyevans/roda-sequel-stack].
Ruby 3.0.1
Elm for view

run with `./run`
